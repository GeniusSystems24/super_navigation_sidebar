super_navigation_sidebar v2.1.0

A themeable, responsive Flutter navigation sidebar. One data model renders in three modes — a full expanded tree with connectors, an icon-only rail with hover flyouts, and an off-canvas drawer. This page is a faithful browser recreation — every control below drives the real widget behavior.

3 layout modes typed NavNode<T> AppBar + Shell RTL / a11y 0 dependencies

account_treePositional roles

Depth-0 leaf = direct, branch = module; nested branch = group, leaf = item. No role field.

lockPermission gating

locked + lockMessage dim the row, block nav, and show the reason on hover — try Wire / SWIFT.

circleStatus dots

NavNodeStatus marks fiscal-period state: open · closed · locked · attention.

boltRail flyouts

In rail mode, hovering a module opens a grouped flyout of its leaves.

starQuick Access

favoritable adds per-row stars and a synthesized favorites band pinned on top.

searchCommand palette

allowSearchDialog — the single switch — puts a search trigger in the pane that opens a full NavSearchDialog. searchable is the separate inline tree filter.
